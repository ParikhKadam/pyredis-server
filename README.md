# pyredis-server
> Python implement redis server.

## description
It's easy to run a redis server for every one.

## Usage
+ download from git;
+ cd the src/;
+ run
```bat
python main.py [[IP] PORT]
```
The default IP is 127.0.0.1, PORT is 6379.
